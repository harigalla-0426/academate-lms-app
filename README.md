# Academate - Learning Management System

Academate is a Learning Management System (LMS) developed as part of the software engineering course project at Indiana University, Bloomington. It provides a platform for seamless learning, collaboration, and management of courses.

## Tech Stack

- Next.js
- MongoDB
- Express.js
- Node.js

## Functionalities

### User Authentication

- Users can register and log in as students, faculty, or administrators.
- Access personalized dashboards based on user type.

### Dashboard

- View all enrolled courses.
- Access course materials, announcements, and assignments.

### Chat Functionality

- Interact with other users within the same course.
- Real-time messaging for effective communication.

### Assignment, Submission, and Grading

- Faculty can create assignments.
- Students can submit assignments.
- Faculty can grade and provide feedback.
